CSS

CSS stands for cascading style sheets. CSS can be added to HTML elements in 3 ways:

  1. Inline - by using the style attriute in HTML elements.
  2. Internal - by using a style element in the head section.
  3. External - by using an external css.

The most powerful way of css in HTML is Inline CSS but we should use External CSS for the application.

Class & ID

  1. Class - The HTML class attriute is used to define equa styles for elements with the same class name. So all HTML elemenents with the same class attribute will get same style.
  2. ID- The id attribute specifies a unique id for an HTML element. The id value can be used by CSS and Javascript to perform certain tasks for the element with the specific id value.
